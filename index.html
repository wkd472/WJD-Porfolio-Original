Playing video using HTML5

This is just an example of playing video using HTML5's video element. Here I am using this element and some JQuery to dynamically load new video sources in response to a button click and then some JavaScript to startup the playback. HTML5 makes it easier to add video to web sites. In the past I've used the VLC plugin to do this. It was a real pain and only worked in FireFox. I found out that IE plays only MP4 files and Firefox only plays ogg and webm files. So I found a neat script at http://www.quirksmode.org/js/detect.html that detects the browser type and based on the browser type I then play the supported video type.

Note : On the left panel of this page under tools there is a link to a utility that detects the type of video support your browser has when using the HTML5 video element.


To watch these videos you need to install the latest version of IE, Chrome, Safari, or Firefox.
